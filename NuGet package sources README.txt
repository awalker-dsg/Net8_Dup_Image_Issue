To build MauiDupImageTestApp, you'll first need to do the following in Visual 
Studio to add the LocalNuGetRepo folder to your NuGet package sources:

1. Select "Tools" > "NuGet Package Manager" > "Package Manager Settings"
2. Select "Package Sources"
3. Click the "+" button
4. Set the name to "DupImageTestSource" and the Source to full folder path to
   this test code's "LocalNuGetRepo" folder.
